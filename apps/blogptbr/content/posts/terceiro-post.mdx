# Bem-vindo ao MDX!
ALTERADO PELO JULES
Este é um post escrito em **MDX**, que permite combinar Markdown com componentes React.

## Exemplo de Componente React

<CustomComponent>
  Este é um componente React personalizado renderizado dentro do MDX!
</CustomComponent>

## Funcionalidades do MDX

- Markdown tradicional funciona normalmente
- Você pode usar **negrito** e *itálico*
- Listas também funcionam
- E ainda pode adicionar componentes React!

### Código

Aqui está um exemplo de código:

```javascript
const exemplo = () => {
  console.log('Olá do MDX!');
```
